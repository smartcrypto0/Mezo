Track Mezo network activity, MUSD metrics, and token data through these official and community resources.

## Official Dashboards

- [MUSD on Mezo (Dune)](https://dune.com/scoffie/musd-on-mezo) - MUSD supply, minting activity, and usage metrics
- [Chain Overview (Dune)](https://dune.com/scoffie/chainoverviewmezo) - Network-level statistics and activity
- [DexScreener](https://dexscreener.com/search?q=0xdD468A1DDc392dcdbEf6db6e34E89AA338F9F186) - Token pairs and trading data

## Aggregators

- [DefiLlama](https://defillama.com/chain/mezo) - Total Value Locked (TVL), stablecoin market cap, and volume
- **CoinGecko:** Coming soon
- **CoinMarketCap:** Coming soon

## Community Owned

Dashboards created and/or maintained by the Mezo community. Reach out in the Discord if you have a dashboard you would like to have added to this page, or create a PR with the addition.

- [MezoTools](https://mezotools.cc/) - Detailed MUSD analytics, system collateralization, individual trove details, MUSD price, redemptions page, and more.
